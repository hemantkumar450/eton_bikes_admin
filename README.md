# Etonbikes

Etonbikes dashboard built on Angular 8

## Clone


```bash
git clone https://github.com/hemantkumar450/eton_bikes_admin.git
```

## Installation

```bash
npm install
```
## Start Local
```bash
npm start
```
## Build
```bash
npm run-script build
```

## Environments

Change environment files while running in production
